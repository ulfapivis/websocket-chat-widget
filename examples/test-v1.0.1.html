WebSocket Chat Widget v1.0.1 Test

About This Demo

This demo shows the WebSocket Chat Widget v1.0.1 in action. It's using a configuration similar to what you would get from the configurator tool.

Notable features in v1.0.1:

  • Fixed icon rendering and sizing issues
  • Improved WebSocket connection stability
  • Enhanced markdown rendering for messages
  • Added support for image and URL-based icons
Open Configurator

Testing Instructions

This demo is wired to a test WebSocket server. You can try the following:

  • Click the predefined questions to see responses
  • Type custom messages and observe the response
  • Test the markdown rendering by sending a message with markdown syntax
  • Notice the custom image-based chat button icon (instead of SVG)

Note: Since this is a demo, the server may have limited responses.
